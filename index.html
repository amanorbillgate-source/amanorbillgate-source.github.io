Welcome

This website is a personal learning project created to practice and improve my skills in front-end web development.

It focuses on the fundamentals of HTML, CSS, and responsive design, with an emphasis on clean layout, accessibility, and mobile-friendly interaction.

The site is built entirely on the front end ie. no backend or database, making it a simple but effective space for experimentation and growth.

Purpose of the Website

  • Practice real-world website structure
  • Experiment with design,and colors
  • Understand navigation and user experience on mobile devices
  • Apply concepts learned from tutorials and documentation

Credits & Learning Sources

Some parts of this project were developed with guidance from online tutorials, documentation, and AI tools such as YouTube resources and GitHub Copilot. These tools were used strictly for learning and skill development.
